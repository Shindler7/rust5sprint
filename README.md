# Поиск ошибок и оптимизация

Учебный проект: проектная работа модуля 5 "Использование инструментов
для отладки и оптимизации кода на Rust".

## Задачи проекта:

- использовать средства тестирования, отладки, профилирования, бенчмарки
  выявить ошибки, проблемные и "узкие" места
- обновить код, устранив выявленные проблемы
- повторить тесты с оптимизированным кодом, получив положительные результаты

## Инфраструктура тестирования

### Miri, Valgrind, Sanitizers, бенчмарки

Информацию об использовании см. в файле [NOTES.md](NOTES.md).

### Профилирование
